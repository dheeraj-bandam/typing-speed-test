# Typing Speed Tester

A simple and interactive web-based **Typing Speed Tester** built with HTML, CSS, and JavaScript.  
Test your typing speed (WPM) and accuracy by typing a given sentence.


## Features

- Displays a sample sentence to type
- Calculates typing speed in Words Per Minute (WPM)
- Calculates typing accuracy in percentage (%)
- Simple and clean user interface
- Easy to extend and customize
